변수와 상수에 대해 배워보자

js 변수와 상수는 타입을 명시하지 않는다(모든 타입의 데이터를 참조할 수 있다.)

변수 var과 let

  • js의 변수는 타입을 명시하지 않는다.
  • 변수는 변할 수 있는 수로 참조하는 데이터를 바꿀 수 있다.
  • var : 윈도우 필드로 정의되고 전역에만 선언 가능하다.
  • let : scope({}블럭을 의미)가 명확한 변수로서 es6 문법이다

상수와 const

처음 선언하면 참조하는 data를 바꿀 수 없는 것을 상수라하고 es6에서 추가되었다 (자료형이나 함수를 선언할때 흔히 사용) 예) Math.PI처럼 변하지 않는 정보
