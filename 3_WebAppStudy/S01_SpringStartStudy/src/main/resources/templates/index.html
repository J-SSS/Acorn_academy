안녕 스프링 부트1

스프링 부트는 자바 웹앱을 MVC 패턴으로 개발하는 프레임워크입니다.

spring-boot-devtools 개발자도그

  • spring 내부의 코드가 변경되면 서버 재시작 및 배포를 진행
  • 크롬 브라우저와 소켓 통신을 하다가 서버 재시작을 하면 브라우저를 새로고침한다
  • 인텔리제이에서 컴파일러 설정에 자동새로고침을 허용해줘야함
  • 크롬 확장 프로그램에서 liveReload를 추가해야한다.
  • https://chrome.google.com/webstore/detail/livereload%2B%2B/ciehpookapcdlakedibajeccomagbfab/related?hl=ko
  • 인텔리제이의 빌드-프로젝트 오토메틱 설정이 체크되어있는지 확인

spring-boot-start-web 웹앱을 개발하는 프레임워크

  • tomcat(10.~)이 포함 (이클립스 재단에서 개발중인 jakarta)
  • 톰캣의 서블릿에 해당하는 컨트롤러(@Controller)를 작성할 수 있음.(요청과 응답 처리)
  • 컨트롤러를 작성할 때 꼭.. 처음 생성된 패키지 하위에다 작성해야한다...!!(componet-scan)
  • Jackson 라이브러리가 포함되어있어서 Json파싱을 편하게 할 수 있다...
  • RESTFull API를 제공.. 다양한 메서드(GET,POST,PUT,DELETE,PATCH), 패스의 파라미터 전송
  • sum.do?a=1&b=10
  • sum.do?a=1&b=10
