길이와 index를 사용하는 자료형 Array

JS에는 java의 Array가 없고 ArrayList와 거의 동일한 자료가 있다.

object와 array의 차이

  • Array는 자료의 순서인 index와 길이가 무조건 존재한다.
  • Object는 key와 value로 이루어진 자료 구조...(==자바의 Map)
  • Object는 인덱스와 길이가 존재하지 않는다.
  • Array의 index를 키로 사용은 하지만, 물리적으로 존재하지 않는 순서다
  • JS에는 key가 순서가 있고 길이가 있는 object를 array로 변환하는 함수가 있다

배열의 필드 접근법

  • list.0 (x)
  • list[0] (o)
  • list.get(index)
